### git 업로드 하는 방법 ~!
1. git init             
#### Git 저장소 초기화
2. git status           
#### 현재 상태 확인
3. git add .           
#### 변경된 파일 전체 
4. stage에 올리기
5. git commit -m "전하고 싶은 메세지지"  
#### 커밋 메시지와 함께 저장
6. git log              
# 커밋 로그 확인
7. git branch name      
# 브랜치 생성
8. git checkout name    
# 브랜치 이동
9. git merge name       
# 브랜치 병합
